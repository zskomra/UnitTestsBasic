CUT - code under test
Jeden test sprawdza tylko jedną część kodu
Testy integracyjne - testuja jak zachowa sie program z baza danych, zewnetrznym serwisem
e2e - testuje sie system z poziomu uzytkownika, sprawdza cale funkcjonalnosci biznesowe(testy manualne żeby przeklikac z graficznego punktu widzenia)

Odpalanie testow w terminalu mvn test , mvn -Dtest=NazwaKlasy test (poprzecinku nazwy klas mozna robic), metode -Dtest=NazwaKlasy#nazwaMetody test
